# Приветствующий Telegram-бот

Данный бот умеет приветствовать пользователя на разных языках и изучать новые слова. 

## Возможности
<ul>
    <li>Ответ на приветствие пользователя и обращение по имени</li>
    <li>Добавление новых приветствий в словарь</li>
    <li>Удаление приветствий из словаря</li>
    <li>Возможность сделать боту комплимент :)</li>
</ul>

## Технологии
<ul>
    <li>JavaScript</li>
    <li>Node.js 22</li>
    <li>telegraf 4.16.3</li>
    <li>dotenv 17.2.4</li>
</ul>

## Использование
<ol>
    <li>Установить зависимости командой <code>npm install</code></li>
    <li>Добавить токен бота в .env файл</li>
    <li>Запустить бота командой <code>node index.js</code></li>
    <li>Ввести в чате бота команду <code>/help</code> и следовать инструкциям</li>
</ol>

## Пример использования
```text
Привет
Привет, user name!!!

hola
Hola, user name!!!

/add hi английский
/rm konnichiwa
```
